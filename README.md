VAYEV Launcher v2.6 - README

Thank you for trying out VAYEV Launcher! This tool was created to solve those annoying Minecraft issues and provide a smoother experience.

Potential Issues & Solutions:

* Issue 1: The application does not open when double-clicked.
  - Solution: On some Linux distributions, you need to grant execution permission. Open your terminal, navigate to the folder, and run: chmod +x "VAYEV Launcher". After that, double-click it again or run ./"VAYEV Launcher" in the terminal.

* Issue 2: The game failed to launch or gave a library error.
  - Solution: Please make sure that Java and the necessary Minecraft dependencies are fully installed on your system.

If you encounter any bugs, feel free to report them to me. Just keep in mind that this is one of my first projects, so please be kind.
